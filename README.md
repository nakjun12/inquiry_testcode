### 🚀 문의사항 작성하기 E2E 테스트

**기간:** 2024년 1월

#### 개요

cypress를 사용하여 E2E 테스트를 직접 만들어보기 위한 프로젝트입니다.

#### 도전한 기술적 과제

- **E2E 테스트:** 폼 작성하기를 cypress를 활용하여 E2E 테스트를 구현하였습니다.
- **MSW:** MSW(Mock Service Worker)를 사용하여 실제 API 호출을 모방하는 테스트 환경을 구성하였습니다.
